# Changelog

Усі помітні зміни в цьому проєкті будуть задокументовані в цьому файлі.

Формат базується на [Keep a Changelog](https://keepachangelog.com/uk/1.0.0/), 
а цей проєкт дотримується [Semantic Versioning](https://semver.org/lang/uk/).

## [0.1.0] - 2026-02-25
### Added
- Скрипт `data_load.py` для завантаження сирих даних про ТЗ за 2024 рік.
- Скрипт `data_quality.py` для генерації Markdown-звіту про якість даних (перевірка дублікатів, пропусків, аномалій).
- Скрипт `data_research.py` для очищення даних та перевірки трьох ключових гіпотез.
- Скрипт `visualization.py` для побудови графіків (паливо, вік ТЗ, популярні бренди) за допомогою `matplotlib` та `seaborn`.
- Файл `README.md` з описом цілей проєкту, джерела даних та списком гіпотез.
