# 📌 Аналіз операцій реєстрації транспортних засобів в Україні (2024)

## Мета: *Побудувати конвеєр обробки та аналізу великих масивів даних щодо реєстраційних операцій ТЗ за 2024 рік. Завдання включає очищення сирих даних, проведення дескриптивного аналізу та візуалізацію аномалій і трендів для підтримки прийняття рішень у транспортній галузі.*

Дослідження спрямоване на:

- аналіз структурних характеристик зареєстрованих транспортних засобів (марка, модель, тип пального, тип кузова, рік випуску);
- оцінку вікової структури транспортних засобів, що проходять реєстрацію;
- ідентифікацію ринкових лідерів та сегментацію за типами: Проведення порівняльного аналізу обсягів реєстрацій легкових авто відносно інших категорій, а також виявлення найбільш затребуваних марок (брендів), що формують основний обсяг ринку;
- виявлення аномалій та патернів реєстрацій

## Джерело даних

Дані отримані з офіційного порталу відкритих даних України: [data.gov.ua](https://data.gov.ua/dataset/06779371-308f-42d7-895e-5a39833375f0)  

## Гіпотези

- ### Гіпотеза 1  
Дизельні транспортні засоби складають більшість серед реєстрацій спеціалізованого транспорту

- ### Гіпотеза 2  
Більшість транспортних засобів, що проходять реєстрацію у 2024 році, мають вік понад 10 років

- ### Гіпотеза 3
Легкові автомобілі складають найбільшу частку реєстрацій, при цьому певні марки демонструють значно вищу частоту реєстрацій
